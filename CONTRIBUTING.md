# How to get started

1. Clone this repo:

   ```sh
   git clone https://github.com/TeenVest/TeenVest.git
   ```

2. Install `npm` packages:

   ```sh
   npm install
   ```

3. Configure a `.env` file using this command:

   ```sh
   cp .env.example .env
   ```

4. Configure `Auth0` in `.env` and get started.
